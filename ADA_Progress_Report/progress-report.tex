\begin{abstract}
\addcontentsline{toc}{chapter}{Abstract}
\thispagestyle{plain}

Efficient route planning is a fundamental problem in transportation networks and graph
algorithms.
Traditional navigation systems optimize routes based on travel distance or time, but these
criteria do not minimize fuel expenditure when prices vary across geographic locations.
This project investigates the fuel-aware route optimization problem: given a road network
with fuel stations at different prices, jointly determine the travel path and the refueling
decisions along that path to minimize total fuel cost while ensuring the vehicle never runs out
of fuel.

The transportation network is modeled as a weighted directed graph in which nodes represent
fuel stations and edges represent feasible travel segments.
We implement and compare six algorithmic strategies: State-Expanded Dijkstra, Greedy,
Dynamic Programming, Standard A*, Refuel A* (RF-A*) of Nandy et al.~(2023), and a
heuristic extension we call Partial-Fill A* (PF-A*).
PF-A* strengthens the RF-A* heuristic to account for the vehicle's current fuel level and the
cheapest reachable station, enabling tighter pruning while preserving optimality.
Admissibility and consistency of the PF-A* heuristic are formally proved.

All algorithms are benchmarked on real Thailand road network data assembled from
OpenStreetMap and EPPO provincial price records, as well as on controlled synthetic graphs.
This is the first empirical study of Gas Station Problem algorithms at Thailand-network scale
with real provincial price data, and PF-A* is the first heuristic for this problem that
conditions on both the vehicle's current fuel level and the cheapest reachable station.
\end{abstract}
